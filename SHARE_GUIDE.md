## 快速开始

1. 克隆项目
   ```bash
   git clone https://github.com/helloforrestwu-ctrl/study-abroad-ai-agent.git
   cd study-abroad-ai-agent
   ```

2. 运行系统
   ```bash
   python3 main.py

   ```bash

1. **准备工作**
   ```bash
   cd study-abroad-ai-agent
   python3 demo.py
   ```

2. **录制工具推荐**
   - macOS: QuickTime Player（系统自带）
   - Windows: Xbox Game Bar（Win+G）
   - 跨平台: OBS Studio（免费）

3. **录制内容建议**（2-3分钟）
   - 0:00-0:15 项目介绍
   - 0:15-1:00 演示评估流程
   - 1:00-2:00 展示推荐结果
