# 快速开始指南 | Quick Start Guide

## 5分钟上手留学规划AI智能体

### 第一步：启动系统

```bash
cd /Users/forrestwu/Desktop/AI/留学规划专家
python main.py
```

### 第二步：回答问题

系统会分步引导您提供信息：

#### 问题1：目标学位和国家
```
请输入学位类型 (本科/硕士/博士): 硕士
请输入国家/地区 (可多选，用逗号分隔): 美国,英国
```

#### 问题2：学术背景
```
学校: 北京大学
专业: 计算机科学
GPA (或输入'计算'让我帮您): 3.6
```

#### 问题3：语言成绩和经历
```
考试类型 (TOEFL/IELTS): TOEFL
分数: 105
经历: AI实验室科研项目; 字节跳动实习; ACM竞赛铜奖
```

### 第三步：查看评估报告

系统会自动生成：
- ✅ 背景竞争力评估（0-100分）
- ✅ 个性化选校清单（冲刺/匹配/保底）
- ✅ 每所学校的匹配理由和截止日期

### 第四步：选择增值服务（可选）

```
请选择服务编号 (1-5):
1. 📅 详细申请时间线规划
2. ✍️  文书素材挖掘与立意建议
3. 🎤 面试准备与模拟
4. 🛂 签证材料清单与攻略
5. ❌ 暂不需要，结束咨询
```

## 常用命令

### 运行系统
```bash
python main.py
```

### 查看项目结构
```bash
tree -L 2
```

### 编辑学校数据库
```bash
# 使用任意文本编辑器编辑
vim knowledge/schools.json
# 或
nano knowledge/schools.json
```

### 查看配置
```bash
cat config/settings.py
```

## 下一步

1. **扩展学校数据**：编辑 `knowledge/schools.json` 添加更多学校
2. **自定义评估标准**：修改 `config/settings.py` 中的评分权重
3. **添加新功能**：在 `agents/` 目录下创建新的Agent
4. **集成API**：在 `utils/` 中添加外部数据源集成
