# 留学规划AI智能体系统 | Study Abroad Planning AI Agent System

## 项目简介

这是一个基于多Agent协作架构的留学规划智能咨询系统，由资深留学规划专家（Commands）总指挥，配备多个专业助手（Agents）协同工作，为学生提供个性化的留学申请方案。

### 核心特点

- **20年行业经验**：基于资深留学规划专家的知识体系
- **多Agent协作**：10个专业Agents分工协作
- **7个国家/地区**：支持美国、英国、加拿大、澳大利亚、香港、新加坡、马来西亚
- **分步引导式**：逐步收集信息，避免信息过载
- **数据驱动**：用数据支撑观点，避免模糊表述

## 系统架构

### Commands (总指挥)
**角色**：留学规划专家
**职责**：
- 分步引导学生提供信息
- 协调各个Agent完成任务
- 整合结果并提供建议

### Agents (专业助手)

#### 核心评估流程
1. **BackgroundEvaluator** - 学生背景评估
   - 评估GPA、语言成绩、科研/实习经历
   - 计算综合竞争力分数
   - 给出优劣势分析

2. **GPACalculator** - GPA计算器
   - 支持4.0制和百分制转换
   - 加权GPA计算
   - 目标GPA规划

3. **SchoolMatcher** - 学校匹配引擎
   - 根据背景智能匹配学校
   - 按冲刺/匹配/保底三档推荐
   - 提供匹配理由和截止日期

4. **ResultValidator** - 结果验证器
   - 验证推荐结果的合理性
   - 检查GPA和语言要求一致性
   - 确保申请梯度合理

5. **ReportGenerator** - 评估报告生成器
   - 生成专业的留学规划报告
   - 包含背景评估、选校清单、申请建议
   - 支持导出为文件

#### 可选增值服务
6. **TimelineAgent** - 时间线规划器
   - 制定详细的申请时间表
   - 标注关键截止日期
   - 提醒常见风险点

7. **EssayAdvisor** - 文书素材挖掘
   - 提供3个文书立意角度
   - 展示核心叙事逻辑
   - 避免陈词滥调提示

8. **InterviewCoach** - 面试辅导
   - 高频面试问题及答题要点
   - STAR法则答题框架
   - 学校文化偏好分析

9. **VisaAdvisor** - 签证辅导
   - 各国签证材料清单
   - 资金证明要求
   - 高频拒签原因及应对策略

## 安装与使用

### 环境要求
- Python 3.7+
- 无需额外依赖库（纯Python实现）

### 安装步骤

```bash
# 1. 克隆或下载项目
cd 留学规划专家

# 2. 直接运行（无需安装依赖）
python main.py

# 或者给予执行权限后直接运行
chmod +x main.py
./main.py
```

### 使用流程

1. **启动系统**
   ```bash
   python main.py
   ```

2. **按步骤回答问题**
   - 第一步：选择目标学位和国家
   - 第二步：提供学术背景（GPA、专业等）
   - 第三步：提供语言成绩和经历

3. **查看评估报告**
   - 系统自动生成背景评估
   - 推荐选校清单（冲刺/匹配/保底）
   - 申请建议

4. **选择增值服务**（可选）
   - 时间线规划
   - 文书素材挖掘
   - 面试准备
   - 签证攻略

## 功能特性

### 1. 智能背景评估
- 多维度评分（GPA 35%、语言 25%、科研 20%、实习 20%）
- 竞争力等级判定（强/中/待提升）

- 每个阶段的核心任务清单

### 4. 文书素材挖掘
- 3个可能的文书立意角度
- 核心叙事逻辑框架
- 避免陈词滥调的具体建议

### 5. 面试准备辅导
- 通用/学术/行为三类高频问题
- STAR法则答题框架
- 目标学校文化偏好分析

### 6. 签证办理攻略
- 7个国家/地区的签证材料清单
- 资金证明详细要求
- 高频拒签原因及应对策略

## 知识库扩展

系统支持自定义扩展学校数据库。编辑 `knowledge/schools.json` 文件即可添加更多学校信息。

### 学校数据格式

```json
{
  "国家名": [
    {
      "name": "School Name in English",
      "中文名": "学校中文名",
      "ranking": 排名数字,
      "tier": "冲刺/匹配/保底",
      "gpa_requirement": 3.5,
      "toefl_requirement": 100,
      "ielts_requirement": 7.0,
      "programs": ["专业1", "专业2"],
      "deadline": "2024-12-15",
      "match_reason": "匹配理由描述"
    }
  ]
}
```

## 使用建议

1. **如实填写信息**：评估结果的准确性取决于输入信息的真实性
2. **GPA计算**：如不清楚GPA，使用内置计算器
3. **多国申请**：建议选择2-3个国家以增加录取机会
4. **时间规划**：尽早使用时间线功能，留足准备时间
5. **文书准备**：利用文书素材挖掘功能，避免模板化写作

## 常见问题 FAQ

**Q: 推荐的学校数据准确吗？**
A: 知识库基于真实院校数据，但具体要求可能变化，建议以官网为准。

**Q: 可以添加更多学校吗？**
A: 可以！编辑 `knowledge/schools.json` 按格式添加即可。

**Q: GPA计算支持哪些格式？**
A: 支持4.0制字母成绩（A+, A, B+等）和百分制（90, 85等）。

**Q: 系统支持博士申请吗？**
A: 支持本科、硕士、博士三种学位的申请规划。

**Q: 评估报告可以导出吗？**
A: 在ReportGenerator中调用export_to_file()方法即可导出为txt文件。
