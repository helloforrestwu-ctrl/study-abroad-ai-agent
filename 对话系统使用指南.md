## 💡 使用技巧

### 技巧1: 提供完整信息
越详细的信息，系统给出的建议越准确：

✅ **好的提问**:
```
我是清华大学计算机科学大四学生，GPA 3.7，托福108，
有2段AI相关科研经历和1段腾讯实习，想申请美国Top 30的CS硕士，
希望侧重AI方向，预算充足。
```

❌ **不够详细的提问**:
```
帮我推荐几所学校
```

### 技巧2: 分步骤咨询
不用一次性问所有问题，可以先评估，再选校，最后优化：

```
第一轮: 先评估我的背景竞争力
第二轮: 根据评估结果推荐学校
第三轮: 针对某几所学校做深入分析
第四轮: 制定申请时间线和策略
```

### 技巧3: 利用系统记忆
系统会记住您之前提供的信息：

```
【第一次】我是北大CS，GPA 3.6，托福100
【系统】[给出评估和选校]

【第二次】如果我把托福刷到105，选校会有什么变化？
【系统】[基于之前信息，只更新托福成绩重新评估]
```

### 技巧4: 请求具体建议
明确告诉系统您需要什么：

```
我需要数据支撑的建议，不要模糊的表述
```

```
请给我3-5个具体可行的提升建议
```

```
请用表格形式呈现选校清单，方便对比
```

## ❓ 常见问题

### Q1: 系统推荐的学校准确吗？
A: 系统基于30+所院校的真实数据和历年录取标准，但最终决策需要：
- 访问学校官网确认最新要求
- 咨询学长学姐或专业中介
- 结合个人偏好和家庭情况

### Q2: 可以保存评估报告吗？
A: 可以！评估报告已经是Markdown格式，您可以：
- 直接复制到笔记软件
- 保存为.md文件
- 转换为PDF

### Q3: 系统会更新数据吗？
A: 建议定期查看：
- 学校官网最新要求
- 最新排名数据
- 当年截止日期

您也可以要求系统："请帮我查询XX学校2025年的最新要求"

### Q4: 可以用于其他专业吗？
A: 可以！系统支持所有专业，只需：
- 在咨询时说明专业
- 系统会根据专业特点调整建议
- 商科、理工科、文科都适用

### Q5: 一次咨询够吗？
A: 建议多轮优化：
- 初步评估后，根据建议提升背景
- 2-3个月后重新评估
- 申请前最后确认选校清单
