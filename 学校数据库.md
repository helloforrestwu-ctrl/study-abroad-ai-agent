## 使用说明

### 如何使用此数据库

1. **根据学生GPA筛选**
   - 学生GPA ≥ 学校要求 → 保底/匹配
   - 学生GPA < 学校要求但接近 → 匹配/冲刺
   - 学生GPA 明显低于要求 → 冲刺（需其他优势）

2. **根据语言成绩筛选**
   - TOEFL/IELTS 达标 → 可申请
   - 差距在5分/0.5分以内 → 可冲刺
   - 差距过大 → 不建议

3. **综合匹配度评分**
   ```
   匹配分数 = 基础分50
            + GPA匹配度(最高20分)
            + 语言匹配度(最高15分)
            + 背景匹配度(最高15分)
   ```

4. **推荐原则**
   - 冲刺: 匹配分数 60-75
   - 匹配: 匹配分数 75-90
   - 保底: 匹配分数 90+
